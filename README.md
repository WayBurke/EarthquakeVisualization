## Web Data Visualization
Repository that demonstrates the use of Leaflets to create Earthquake Visualization using data sourced from the United States Geological Survery (USGS). The visualized USGS data will allow them to better educate the public and other government organizations on issues facing our planet.

# MODULE 15 CHALLENGE - Web Data Visualization - Leaflet Challenge

## Overview
The purpose of this project is to use JavaScript Leaflet to create web data visualization of the USGS earthquake dataset that was accessed via a GeoJSON Object. In addition to apply Tectonic Plates found at https://github.com/fraxen/teconicplates to plot additional data to the map.

## How to use
By clicking on the active deployment page, you will be taken to a landing page for which you can select from either image. Each image represents the Parts of this project. 
* Part 1: Create the Earthquake Visualization
  
* Part 2: Gather and Plot More Data (optional)

## Files and Resources used
* JavaScript code for the manipulation of the HTML file
* HTML files used to launch the webpages illustrating the map visualizations.
* MapBox for the Satellite imagery with the Tectonic Plates
* OpenStreetMap for the Street and Topographic Map
